# Acknowledgements

This repository is prepared as an official-style release for Denoising-Driven
Learning (DDL). We thank the authors of the following projects and baselines:

- **X-Restormer**, the vanilla restoration baseline used in the paper for DDL
  experiments.
- **NAFNet / Restormer-style restoration backbones**, which inspired the clean
  residual restoration network implementation.
- **IR-SDE and score-based restoration works**, which motivated the diffusion
  denoising view used by DDL.
- **PromptIR, AirNet, DA-CLIP, ResShift, UniRestore, EAMamba, DeepGSR, and
  FDIN**, which are important comparison methods in unified and generalizable
  image restoration.
- The maintainers of the public datasets listed in
  [`DATASETS.md`](DATASETS.md).

If you use this code, please also cite the original datasets and baseline
methods that are relevant to your experiment.
